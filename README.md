# Автобиография — Персональный сайт-портфолио
Персональный многостраничный сайт-визитка с интерактивными элементами и адаптивным дизайном.
## Описание
Сайт включает в себя:
- Главную страницу с краткой информацией и статистикой посещений
- Страницу "Обо мне" с биографией и модальным окном
- Страницу "Навыки" с простым калькулятором и конвертером валют
- Страницу "Портфолио" с фильтрацией проектов и сортируемой таблицей
- Страницу "Контакты" с валидацией формы обратной связи
## Технологии
- HTML5 — семантическая разметка
- CSS3 — адаптивный дизайн
- JavaScript (ES6+) — интерактивность
- Page Object Model (POM) — архитектура JavaScript-кода
- Fetch API — асинхронные запросы
- LocalStorage/SessionStorage — хранение данных
## Установка и запуск
### Необходимые требования:
- Visual Studio Code (или другой редактор кода)
- Браузер (Chrome, Firefox, Safari, Edge)
### Пошаговая инструкция:
1. Скачайте проект

```bash
git clone https://github.com/MrTvisti/Autobiography_JS_Website.git
cd Autobiography_JS_Website
```

*Или скачайте архив с GitHub и распакуйте его.*

2. Установите расширение Live Server
- Откройте Visual Studio Code:
- Перейдите во вкладку Extensions (Расширения) или нажмите Ctrl+Shift+X
- В поиске введите Live Server
- Найдите расширение от Ritwick Dey
- Нажмите Install (Установить)
3. Запустите проект
- Откройте папку с проектом в VS Code
- Найдите файл index.html
- Нажмите правой кнопкой мыши на файл
- Выберите "Open with Live Server" (Открыть с Live Server) [*Или нажмите кнопку Go Live в нижней панели VS Code.*]
***
### Сайт автоматически откроется в браузере по адресу http://127.0.0.1:5500/index.html
